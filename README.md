# Smart Focus Detector

Project mikrokontroler untuk mendeteksi tingkat fokus belajar mahasiswa
menggunakan sensor dan Arduino.

Smart Focus Detector adalah sistem berbasis mikrokontroler yang bertujuan
untuk membantu mengetahui tingkat fokus belajar berdasarkan data dari sensor.
Project ini dibuat sebagai bagian dari tugas UAS mata kuliah Mikrokontroler
Semester 5.

## Komponen yang Digunakan
1. Sensor Ultrasonik HCSR04
2. LCD 16x2 I2C
3. Push Button
4. Buzzer
5. Breadboard
6. Resistor 220 ohm
7. Resistor 10.000 ohm
8. Potensiometer 10.000 ohm
9. LED (Merah, Kuning, Hijau)
10. Arduino Uno
